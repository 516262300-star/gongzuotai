# ERP 自动上架

## 项目目标

从 ERP 或商品资料源读取商品信息，校验标题、图片、价格、库存、类目和属性后，生成可上架清单或执行自动上架。

## 当前状态

- 状态：规划中
- 自动化脚本：未创建
- 外部文档：待补充

## 数据流

```text
ERP 商品资料 -> 字段清洗/校验 -> 平台字段映射 -> 上架草稿或发布 -> 日志/复核
```
